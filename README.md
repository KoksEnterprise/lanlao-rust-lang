# Lanlao Rust Language

## Project Documentation

### Overview
Lanlao is a programming language inspired by Rust, designed for systems programming with a focus on safety and concurrency.

### Features
- Memory safety without garbage collection
- Seamless concurrency support
- High performance comparable to C/C++

## Usage Instructions

### Installation
To install Lanlao, run the following command:

```bash
cargo install lanlao
```

### Basic Syntax
Here’s a simple example of a Lanlao program:

```lanlao
fn main() {
    println!("Hello, World!");
}
```

### Running the Program
You can run your Lanlao program by using the command:

```bash
lanlao run your_program.lan
```
